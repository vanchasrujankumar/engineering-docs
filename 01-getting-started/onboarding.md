# Onboarding Guide

## Week 1: Setup

- [ ] GitHub access granted
- [ ] GitBook access configured
- [ ] Repo cloned locally
- [ ] Read through Getting Started section

## Week 2: Contribute

- [ ] Make a small doc edit (fix a typo, update a link)
- [ ] Submit your first PR
- [ ] Attend docs review meeting

## Week 3: Ownership

- [ ] Assigned a section to maintain
- [ ] Review at least one PR from another contributor
- [ ] Propose one improvement to the docs structure
