# Quick Start

## Prerequisites

- GitHub account with access to the organization
- GitBook account (SSO via GitHub)

## 5-Minute Setup

1. **Clone the repo**
   ```bash
   git clone https://github.com/vanchasrujankumar/engineering-docs.git
   cd engineering-docs
   ```

2. **Open in GitBook**
   - Visit the GitBook site
   - Log in with GitHub
   - You can edit any page directly in the browser

3. **Make your first edit**
   - Navigate to any page
   - Click "Edit"
   - Make changes
   - Click "Save" — GitBook creates a PR automatically

4. **Get it reviewed**
   - Tag a team member on the PR
   - Once approved, it merges and deploys automatically
